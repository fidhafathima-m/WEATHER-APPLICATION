A responsive weather application built using React + Vite that allows users to search for any city and view real-time weather information using the OpenWeather API.

[Live](https://weatherapplication-eosin.vercel.app/)

## Features

- Search weather by city name
- Displays temperature
- Shows weather condition (Cloudy, Rainy, Sunny, etc.)
- Humidity information
- Wind speed
- Real-time API data
- Fully responsive design
- Deployed on Vercel


## Getting Started

1. Clone the repository

```bash
git git clone https://github.com/fidhafathima-m/WEATHER-APPLICATION.git
cd WEATHER-APPLICATION
```

2. Install dependencies

```bash
npm install
```


### Running the App

1. Start development server
```bash
npm run dev
```

The server will start on http://localhost:5173/
